week1_day3

Week1day3
鉴于已完成平台搭建等准备任务，今天对基本任务作了了解，并学习了西瓜书第一章的内容
明日计划:完成数据集的处理（填补缺失值，标准化，归一化，数据抽样等）和划分（训练、测试、验证）部分的学习并进行一定的实操。

*附本周任务一览

  1. 学习并安装使用Linux
    • 根据自己的需求和喜好选择合适的发行版。推荐Ubuntu或manjaro，有余力的同学可以试试Arch Linux
    • 学习基本Linux命令的使用，能熟练使用命令进行各种文件操作、进程管理（cd/ls/pwd/mv/mkdir/rm/cp/ps/top/htop/kill/less/cat/grep/alias/watch/nvidia-smi），尽量记一些常用的，记不住的可以以后慢慢熟悉
    • 学习使用自己发行版的包管理工具
    • 学习vim的使用方法
    • 推荐大家以Linux为以后的工作环境
    • 简单了解一些ssh和网络协议有关的知识(以及在linux上的相关配置)，今后会经常用到远程服务器来训练模型。

  2. 学习使用Github
    • 推荐教程: 廖雪峰
    • 配置git环境
    • 注册自己的GitHub账号并将以后的任务同步到GitHub，本任务周至少push3次

  3. 配置工作环境
    • 配置好python环境(推荐anaconda)
    • 配置python开发环境。推荐vscode，jupyter notebook/lab以及pycharm，选一个即可
    • 配置CUDA, cuDNN，安装pytorch或tensorflow,有GPU的安装GPU版本

  4. 快速入门学习markdown
    • 学习基本语法
    • 学习使用内联公式(和LaTeX的语法基本一致，某些地方有些小区别)
    • 有余力可以学习LaTeX

机器学习の基本算法
• 参考数据集：iris、Pima Indian
• 参考书籍：周志华《机器学习》，李航《统计学习方法》
• 模型建议进行可视化和数据分析，结果分析等等
基础任务
• 实现对数据集的处理（填补缺失值，标准化，归一化，数据抽样等）和划分（训练、测试、验证）
• 实现K-means聚类，并了解其他常用的聚类算法
• 理解并实现Logistic回归，并尝试多种优化方法（各种梯度下降、牛顿法、Adagrad、Adadelta、RMSprop、Adam等）
• 实现CART Decision Tree
• 了解ID3、C4.5决策树的原理
提高任务
• 理解并实现Random forest【了解集成学习方法】
• 阅读论文：AlexNet
要求
• 不要使用Scikit Learn等机器学习库
• 不要抄别人的代码（大到架构，小到接口），尽量看着公式自己写
• 了解每个算法的优缺点、对数据的要求和常见应用
• 每个算法都要在数据集检验。在一个数据集上检验即可。
以下为建议完成：
1.科学上网 并且学会使用Google
• 服务器请自行购买
• 推荐GitHub学生包里送的50刀digital ocean，相当于免费用10个月(谁用谁知道)
• 或者可以用aws，绑定信用卡可以免费一年
• 利用服务器搭梯子
• 学会尽量用搜索解决自己遇到的问题
• 习惯使用英文进行搜索
2.改善python代码风格
• 参考Google开源项目风格指南并在后面的编程任务中坚持这种代码风格
• conda/pip install autopep8
3.推荐的一些奇奇怪怪♂的东西
• 推荐书籍《统计学习方法》《机器学习》(两本推荐入手或找PDF)
• 推荐课程 CS231n, 网易云课堂吴恩达dl课程
• Guake
• oh my zsh/fish
• 某写满了丰富教程的的博客
• tmux
4.提前学习自己感兴趣的知识
• 推荐了解的ml算法(可以先泛泛地浏览一下，后面的任务都会涉及(不是这一周全部实现))
o KNN
o Logistic Regression
o DecisionTree
o SVM
o boosting
o bagging
o KMeans
o PCA
o …
5.搭建自己的博客
• 使用Github pages + 博客框架（如hexo），搭建自己的博客
• 大家尽量养成写博客的习惯，来记录自己学到的东西。*
